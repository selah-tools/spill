# Spill — Brand Identity

## Name

**Spill**

Domain: **spill.cards** ✅ Available

---

## The Origin

Named using expert brand naming methodologies from Lexicon, Eat My Words, Catchword, and Igor. The process tested 50+ candidates against:

- **SMILE & SCRATCH Framework** (Alexandra Watkins)
- **Sound Symbolism** (Lexicon/Placek)
- **The Naming Matrix** (Abstract → Descriptive × Real-Word → Coined)
- **Marty Neumeier's 8 Criteria**
- **Catchword's 10 Touchstones**

The reference anchor was **Jackbox.tv** — a 2-syllable compound with suggestive meaning, memorable plosives, and a category-defining TLD.

## Visual Direction

### Keywords

- Fluid typography (spilling, flowing motion)
- Card-dealing animation
- Warm but modern palette
- Slightly edgy but safe
- Honest, not church-program

### Color Palette (Suggested)

| Role      | Color         | Hex                    |
| --------- | ------------- | ---------------------- |
| Primary   | Amber/Ember   | `#F59E0B` or `#D97706` |
| Secondary | Deep teal     | `#0D9488`              |
| Neutral   | Warm cream    | `#FEF3C7` or `#FFFBEB` |
| Dark      | Soft charcoal | `#1F2937`              |

**Rationale:** Warmth + modern edge. Amber = glow, spark, ember. Teal = depth, trust.

### Motion Language

- Cards dealing/spilling from a deck
- Liquid flow transitions
- "Spilling" text effects (not literal, suggested)

### Frontend Styling Note

For the MVP, the visual system should be implemented with **plain CSS + CSS variables** rather than UnoCSS. The brand depends on a custom, warm, fluid feel, and the interface only needs a small number of distinct components. A handcrafted stylesheet will make it easier to preserve a strong identity without drifting into generic utility-first UI.

**Best fallback options if the stack changes later:**

- **Tailwind CSS** for speed and ecosystem
- **Panda CSS** for a token-driven design system
- **CSS Modules** if component-level scoping becomes necessary

## Risk Assessment

### The "Tea" Connotation

**Concern:** "Spill" = "spill the tea" = gossip/drama culture.

**Mitigation:**

1. **Redemption arc:** Take something associated with gossip → use it for honest, faith-centered sharing
2. **Biblical anchor:** "Spill your heart before God" — Psalms are pure unfiltered spill
3. **Edge = spread:** Slightly viral language helps 25-year-olds invite friends

**Verdict:** Acceptable risk. The edginess is a feature, not a bug.

### Trademark Checklist

- [ ] USPTO search: "Spill" in Class 9 (software), Class 41 (entertainment)
- [ ] App Store search: Existing "Spill" apps
- [ ] Common law search: Active use in conversation/QA space

**Note:** "Spill" is a common word = weaker protection (harder to defend, easier to use).
